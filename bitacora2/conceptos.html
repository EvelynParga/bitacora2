JavaScript Conceptos basicos

JavaScript

JavaScript (abreviado comunmente JS) es un lenguaje de programacion orientado a objetos, dialecto del estandar ECMAScript.

Insertar JavaScript en HTML

Hay dos formas de incluir JavaScript en HTML y hacer que funcionen juntos: Agregar JavaScript directamente a un archivo HTML, agregar codigo JavaScript en un archivo separado.

Agregar JavaScript directamente a un archivo HTML

Para insertar JS de forma directa en HTML se utiliza la etiqueta -script- -/script-, el codigo JS debe quedar dentro de esta etiqueta.

El codigo JS se puede colocar dentro de la etiqueta -head- o la etiqueta -body-

Dependiendo de donde se agregue el codigo JS en el archivo HTML. La carga sera diferente. Por lo genral se recomienda agregarlo en el -head- para que permanezca separado del contenido HTML. Pero colocarlo dentro de -body- puede ayudar a mejorar la velocidad de carga, ya que el contenido del sitio web se encargara mas rapido, y solo despues de eso procesara el JavaScript.

Agregar codigo JavaScript a un archivo separado

Insertar JS en HTML directamente no siempre es la mejor opcion. Sobre todo si algunos scripts deben utilizarse en varias paginas, por lo tanto, es mejor mantener el codigo JS en archivos separados son:

    La legibilidad y el mantenimiento del codigo es mucho mas facil.
    Los archivos JavaScript en cache mejoran el rendimiento general del sitio web al disminuir el tiempo que tardan las paginas en cargarse.

getElementById()

El metodo getElementById() devuelve el elemento que tiene el atributo ID con el calor especificado. Este metodo es uno de los metodos mas comunes en el DOM de HTML y se usa casi cada vez que desea manipular u obtener informacion de un elemento de su documento. Se vuelve nulo si no existen elementos con el ID especificado. Una identificacion debe ser unica dentro de una pagina. Sin embargo, si existe mas de un elemento con el ID especificado, el metodo getElementById() devuelve el primer elemento en el codigo fuente.

Variables

Las variables de JavaScript son contenedores para almacenar valores de datos. Todas las variables de JavaScript deben identificarse con nombres unicos. Estos nombres unicos se llaman identificadores. Los identificadores pueden ser nombres cortos (como x, y) o nombres mas descriptivos (edad, suma, volumen total).

var, let y const para crear variables. var, abreviatura de variable, es una palabra clave de JavaScript que crea, o declara, una nueva variable. var edad;// para declarar la variable. edad // para llamar a la variable var personajeFavorito= "Bob Esponja";

La palabra clave let indica que la varible se puede reasignar a un valor diferente. Debemos tener en cuenta al usar let (o var) que podemos declarar una variable sin asignar un valor a la variable. En tal caso, la variable se inicializara automaticamente con un valor de indefinido.

La palabra clave const tambien se introdujo y es la abreviatura de la plabra constante. Al igual que con var y le permite almacenar cualquier valor en una variable const. La forma en que declara una variable const y le asigna un valor sigue la misma estructura que let y var. Sin embargo, una variable const no se ouede reasignar porque es constante. Si se intenta reasignar una variable const, se obtiene un TypeError.

Reglas

Las reglas generales para construir nombres para variables (identificadores unicos) son:

    Los nombres deben comenzar con una letra.
    Los nombres pueden contener letras, digitos, guiones bajos y signos de dolar.
    Los nombres tambien pueden comenzar con $ y_.
    Los nombres distinguen entre mayusculas, minusculas (y e Y son variables diferentes)
    Las palabras reservadas (como las palabras clave de JavaScript) no se pueden usar como nombres.

Operadores aritmeticos

JavaScript tiene un amplio conjunto de operadores, entre los que se incluyen operadores aritmeticos, logicos, bit a bit y de asignacion, asi como algunos operadores varios. Los operadores permiten realizar calculos complejos y tomar decisiones logicas en funcion de comparaciones y otros tipos de condiciones. Los operadores aritmeticos estandar son adicion o suma (+), sustraccion o resta(-), multiplicacion (*), y division (/), incremento (++), decremento (--).
